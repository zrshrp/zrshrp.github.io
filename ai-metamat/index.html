ИИ помог разработать метаматериал

Главная / Физика  

В Калифорнийском технологическом институте с помощью искусственного интеллекта с первой попытки создали новый материал. Об этом сообщает Wiley online library.

Ранее для создания нового материала с необходимыми свойствами ученым приходилось использовать метод “проб и ошибок”.

Чтобы создавать необходимые материала значительно быстрее без необходимости проведение множества экспериментов, специалисты привлекли к этому процессу искусственный интеллект. Он использует процесс машинного обучения для предсказания свойств будущих материалов и программирования их структуры.

Так у специалистов с первого раза получился новый материал, обладающий невероятной сжимаемостью и прочностью.

Ученые хотят создать материал, большую площадь которого можно сжать в небольшой объем. По их словам, тогда велосипеды, зонтики и другие вещи можно было бы сложить в карман.

Специалисты отмечают, что на данном этапе исследования важен не созданный ими материал, а возможность “быстрого” его создания с помощью машинного обучения.
