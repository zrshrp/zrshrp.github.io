Надувной камуфляж поможет маскировать объекты и показывать карту рельефа

Главная / Физика  

Инженеры из Корнельского университета и Лаборатории биологии моря разработали метод создания надуваемых объемных структур произвольной формы. Они предлагают использовать такую технологию в том числе для создания правдоподобного рельефного камуфляжа. Статья опубликована в журнале Science.

Для камуфлирования используются разные технологии, одна из самых старых и популярных — специальная раскраска, которая совпадает с основными цветами окружающей среды или другим образом затрудняющая восприятие объекта со стороны. Но большие по площади поверхности все равно могут выделяться за счет неестественности рельефа, поэтому также применяется объемные элементы — например, специальные костюмы для снайперов, состоящие из веток или травы, характерных для конкретной местности.

Новая технология американских инженеров позволяет создавать надуваемые объемные структуры заданной формы, которые можно в дальнейшем использовать для маскировки техники и сооружений. Изначально они представляют собой плоскую поверхность, состоящую из двух слоев — силиконового эластомера и слабо растягивающейся сетки. Для того, чтобы придать конечной структуре конечную форму, необходимо удалить часть сетки на поверхности с помощью лазерного луча. Эта поверхность служит верхней частью воздушной камеры.

При надувании участки поверхности без сетки расширяются гораздо больше, чем эластичные области, подобно тому, как ведет себя воздушный шарик, если стянуть его ниткой. Таким образом, вырезая определенные жесткие участки можно регулировать конечную форму надутого объекта.

Инженеры продемонстрировали несколько вариантов использования технологии. Для примера они создали поверхности, которые при надувании превращаются в реалистично выглядящие камни, а также похожую на суккулент конструкцию. Помимо этого разработчики предлагают и другие применения, к примеру, в сочетании с проектором такую поверхность можно превратить в интерактивный дисплей для отображения трехмерной карты рельефа местности.
