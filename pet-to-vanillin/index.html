Бактерии превратят пластик в ванилин

Главная / Физика  

Решение проблемы пластикового мусора идет по трем направлениям, пластик можно сделать быстро разлагаемым, научиться эффективно его перерабатывать или же извлекать из полимеров полезные вещества. Ученые из Эдинбургского университета в Шотландии пошли по третьему пути и вывели особый вид бактерий E. Coli. Это детище синтетической биологии превращает продукты переработки ПЭТ в весьма востребованный ванильный ароматизатор.

Суммарный объем изделий из ПЭТ в мире достигает 50 млн тонн, которые автоматически превращаются в мусор. Если заниматься только его прямой переработкой, создавать из б/у-изделий материал для новых, ничего принципиально не изменится. Шотландские ученые предложили превратить пластик в источник нового сырья. Для этого они первым делом извлекли из ПЭТ один из компонентов под названием терефталевая кислота.

Потребовалось немало усилий по модификации бактерий для того, чтобы химические реакции при поглощении терефталевой кислоты протекали с должной точностью. В итоге им удалось достичь превращения 79 % исходного вещества в чистый ванилин. Это не только пищевой компонент, но и ценный промышленный химикат, который есть в гербицидах, краске, косметике, чистящих средствах, пеногасителях и т.д. Потребность в ванилине измеряется десятками тысяч тонн в год, поэтому пластик может стать удобным источником этого вещества.
